# 🚀 DragX Telegram Bots

Welcome to **DragXTeleBots** — A collection of powerful, scalable, and production-ready Telegram bots built with ❤️.

## 🤖 About

DragXTeleBots focuses on building:

- ⚡ Userbots
- 🎵 Music Bots
- 🎴 Waifu / Anime Bots
- 🔍 Utility Bots
- 🛠 Automation Tools
- 🌐 Web-integrated Telegram Apps

All projects are designed with:
- Clean architecture
- Modular plugins
- Scalable structure
- Production deployment support

## 🛠 Tech Stack

We primarily use:

- 🐍 Python (Pyrogram / python-telegram-bot)
- 🌐 FastAPI / Flask
- 🗄 MongoDB
- ☁️ AWS / VPS / Docker
- 🔧 GitHub Actions (CI/CD)
